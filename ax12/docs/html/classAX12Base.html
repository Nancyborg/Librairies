Detailed Description

Abstract class used to communicate with an AX12 servomotor.

Subclasses must provide an implementation of the readBytes(), writeBytes() and flushInput() methods.

No error

AX12_COMM_ERROR_CHECKSUM 

The checksum of the status packet is incorrect

AX12_COMM_ERROR_TIMEOUT 

There was a timeout reading the status packet

AX12_COMM_ERROR_FORMAT 

The received status packet was malformed

AX12_COMM_ERROR_ID 

The status packet ID did not match the one in the sent instruction packet

AX12_COMM_ERROR_SYSTEM 

A system error occured

Enumerator

The voltage is out of the operating voltage range

AX12_ERROR_ANGLE_LIMIT 

The Goal Position is set outside the range between CW limit and CCW limit

AX12_ERROR_OVERHEATING 

The internal temperature of the AX12 is outside of the operating temperature range

AX12_ERROR_RANGE 

The instruction sent is out of the defined range

AX12_ERROR_CHECKSUM 

The checksum of the instruction packet is incorrect

AX12_ERROR_OVERLOAD 

The specified maximum torque can't control the applied load

AX12_ERROR_INSTRUCTION 

An undefined instruction was sent
